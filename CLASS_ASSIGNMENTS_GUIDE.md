# 📚 Руководство по системе заданий RezoFlow

## Как работает система заданий по классам

RezoFlow теперь поддерживает умную систему распределения заданий по классам! Учителя могут создавать задания для конкретных классов, и только ученики этого класса их увидят.

---

## 🎓 Для учеников

### Регистрация

1. При регистрации выберите роль **"Школьник"**
2. После выбора роли появятся дополнительные поля:
   - **Номер класса**: выберите от 1 до 11
   - **Буква класса**: выберите от А до Е

Пример: если вы учитесь в 8А классе, выберите **8** и **А**

### Получение заданий

- После регистрации вы будете автоматически получать задания от учителей для вашего класса
- Когда учитель создаст задание для вашего класса, вы получите:
  - 🔔 **Уведомление** в реальном времени
  - 📝 **Задание** в вашем списке задач
  - ⏰ **Напоминания** о приближающихся дедлайнах

### Ваш класс отображается

- На главной странице под логотипом RezoFlow
- В вашем профиле

---

## 👨‍🏫 Для учителей

### Создание задания

1. Нажмите кнопку **"Создать задание"**
2. Заполните информацию о задании:
   - Название
   - Предмет
   - **Класс** (выберите из выпадающего списка: 1А-11Е)
   - Срок сдачи
   - Описание

3. После создания задания:
   - Все ученики выбранного класса **мгновенно** получат уведомление
   - Задание автоматически появится в их списке
   - Ученики других классов **не увидят** это задание

### Пример использования

**Задание**: Решить задачи по алгебре №15-20  
**Класс**: 8А  
**Результат**: Только ученики 8А класса получат это задание

---

## 🔄 Как это работает технически

### Фильтрация по классам

```javascript
// При создании задания учитель выбирает класс
Задание создано для класса: "8А"

// Система отправляет событие со всей информацией
Событие: new-teacher-task
Данные: {
  title: "Решить задачи по алгебре",
  subject: "Математика",
  assignedClass: "8А",  // ← Важно!
  deadline: "2026-04-30"
}

// Каждый ученик получает событие, но обрабатывает только своё
Ученик 8А: ✅ Задание добавлено
Ученик 7Б: ❌ Задание пропущено (не мой класс)
Ученик 8Б: ❌ Задание пропущено (не мой класс)
```

### Уведомления

- **Мгновенные**: используется система Custom Events
- **Умные**: отфильтрованы по классам
- **Персонализированные**: показывают название, предмет и срок

- ✅ Не видят задания других классов
- ✅ Мгновенные уведомления
- ✅ Система напоминаний о дедлайнах

## 💡 Советы

**Для учителей:**
- Всегда проверяйте выбранный класс перед созданием задания
- Используйте понятные названия заданий
- Указывайте реалистичные дедлайны

**Для учеников:**
- Проверяйте уведомления регулярно
- Убедитесь, что выбрали правильный класс при регистрации
- Если не получаете задания - проверьте свой класс в профиле

## ❓ Часто задаваемые вопросы

**Q: Можно ли изменить свой класс после регистрации?**  
A: В текущей версии нет, но это планируется в будущем обновлении.

**Q: Что если я учитель и хочу дать задание всем классам?**  
A: Пока нужно создавать отдельное задание для каждого класса.

**Q: Могу ли я видеть задания других классов?**  
A: Нет, система показывает только задания вашего класса для безопасности и удобства.

**Q: Учитель дал задание, но я его не получил. Что делать?**  
A: Проверьте:
1. Правильно ли указан ваш класс в профиле
2. Совпадает ли он с классом, для которого учитель создал задание
3. Есть ли уведомления в колокольчике

## 🚀 Быстрый старт

### Ученик
1. Зарегистрируйтесь как "Школьник"
2. Выберите свой класс (например, 8А)
3. Ждите заданий от учителей
4. Получайте уведомления

### Учитель
1. Зарегистрируйтесь как "Учитель"
2. Нажмите "Создать задание"
3. Выберите класс из списка
4. Заполните детали и создайте
5. Ученики мгновенно получат уведомление!
